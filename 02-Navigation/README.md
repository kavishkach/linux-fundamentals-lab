# Linux Navigation Commands

Linux navigation commands are used to move through the file system, identify the current location, and view files and directories. These are the first commands every Linux user should learn because they are used in almost every Linux task.

## Commands

| Command        | Description                                                                            |
| -------------- | -------------------------------------------------------------------------------------- |
| `pwd`          | Display the current working directory.                                                 |
| `ls`           | List files and directories in the current directory.                                   |
| `ls -l`        | List files and directories with detailed information (permissions, owner, size, date). |
| `ls -la`       | List all files, including hidden files, with detailed information.                     |
| `cd Documents` | Change the current directory to `Documents`.                                           |
| `cd ..`        | Move to the parent directory.                                                          |
| `cd ~`         | Return to the current user's home directory.                                           |
| `cd -`         | Switch back to the previous working directory.                                         |
| `tree`         | Display the directory structure in a tree format.                                      |

## Learning Outcome

* Navigate the Linux file system.
* Identify the current working directory.
* List files and directories.
* Display hidden files and detailed file information.
* Move between directories using different `cd` options.
* View the directory structure using `tree`.
